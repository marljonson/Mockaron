# Mockaron
A cross-platform clinical portal.

Tech stack: `React` `Android Studio` `Kotlin` `Node.js` `Express` `MongoDB` `Appium` `GitHub Actions`
- Developed a responsive healthcare client interface featuring a React web portal dashboard and a native Android app built in Kotlin
- Constructed a Node.js backend using MongoDB to manage patient-doctor scheduling states and secure user profiles
- Engineered an automated UI testing suite using Appium and Python to achieve 85% code coverage across registration workflows
- Deployed a CI workflow via GitHub Actions to automate linting, bundle building, and rapid test sweeps on every pull request
